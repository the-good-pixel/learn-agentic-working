# Ch. 11 — Context-First Exercise

Drop `meeting-notes-raw.docx` in front of your agent and ask: *"extract action items, owners, and deadlines from this meeting transcript"*. The point: handing the agent a file beats describing it from memory — the agent reads the messy reality (a long Word transcript with action items buried mid-sentence, ambiguous owners, and negotiated deadlines) and pulls structure out of it, instead of relying on whatever you happened to remember.
