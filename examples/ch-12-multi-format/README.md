# Ch. 12 — Multi-format Output Exercise

A single source document (a fictional Brightline Foods Q3 update) for practicing the "same content, four different output forms" exercise from Ch. 12. The cardinal question of the chapter — *artifact, or action?* — is the point.

## Exercise

> Take `quarterly-update-source.docx` and produce four versions of it:
>
> 1. A one-page self-contained HTML report I can open in any browser.
> 2. A print-ready PDF.
> 3. A draft post for an internal channel — Linear / Notion / Slack — with channel-appropriate tone, not a copy-paste of the source.
> 4. A 30-second voiceover script if I were going to record a Loom.
>
> Then tell me which of the four should be *"agent creates the actual ticket / post / file via MCP"* vs. *"agent hands me the markdown and I paste it in"*. Explain why for each.

**You'll know it worked when** the four outputs aren't just four cosmetic skins of the same text — the Slack/Linear/Notion version is shorter and skimmable, the voiceover is something you can read aloud in 30 seconds without tripping, and the agent has an opinion (not a list) on which format is artifact and which is action.
