# Ch. 8 — Persona Inhabit Exercise

Three fictional personas you can step into for the Ch. 8 CLAUDE.md interview exercise — useful if your own working context feels too vague to brief the agent on yet, or if you want to see how role-specific a good `CLAUDE.md` ends up being.

## The exercise

1. Pick one persona and read it.
2. Open a fresh agent session and run the Ch. 8 interview prompt **as if you were them**:

   > *"Interview me for 5 minutes about who I am, what I work on, and what I care about. At the end, write me a CLAUDE.md that captures it."*

3. Answer in the persona's voice. Reference the same tools, the same conventions, the same dislikes.
4. Read the resulting `CLAUDE.md`. Notice how different it looks from one persona to the next.

## You'll know it worked when

- Mira's `CLAUDE.md` enforces no-exclamation-marks and brand voice rules in plain language.
- Hugo's mentions "read `brand-voice.md` before drafting any client copy" as a hard rule.
- Pria's gates Xero posts and pins USD as default.

The shape of a good `CLAUDE.md` is *role-specific*, not generic. That's the point.
